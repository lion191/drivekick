# DriveKick 🚗⚽

DriveKick helps users find the fastest route to sports stadiums and discover nearby parking.

## Features

- Google Maps integration
- Autocomplete stadium search
- Real-time directions
- Nearby parking suggestions
- User location routing

## Tech Stack

- React
- Google Maps API
- Places API

## How to Run

1. Clone repo
2. Add .env file with API key
3. Run npm install
4. Run npm run dev
